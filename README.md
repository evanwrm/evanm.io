# evanm.io

Portfolio website running on [Next.js](https://nextjs.org/) + [Strapi](https://strapi.io).

## Getting Started

First, run the development server:

```sh
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
